# Sound Brief

Upload an audio file and get:

- a transcript
- a structured listening brief
- speaker notes
- soundscape notes
- an event timeline

## Setup

Create `.env.local` with one of:

```env
GEMINI_API_KEY=your_key_here
```

or

```env
GOOGLE_API_KEY=your_key_here
```

## Run

```bash
npm run dev
```

Open `http://localhost:3000`.
